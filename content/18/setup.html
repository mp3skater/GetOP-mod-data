Setup

Installing EarlyGame is a simple process. No, really, it'll take you less than 5 minutes (depending on your download speed, of course). Here's how.

Dependencies

In order for EarlyGame to work, though, you may need some dependencies. All of the links below direct you to Modrinth.

Required Optional
Fabric API Mod Menu (in-game configuration)
Cloth Config API (for 1.18+) Roughly Enough Items (new recipes)

Traditional method

The tried and true formula of simply downloading the .jar file and putting it in the mods folder.

Step 1. Assuming you have Minecraft installed, you need to install the Fabric modloader. Instructions can be found on Fabric's website.

Step 2. Next, download the Fabric API (and the Cloth Config API for 1.18+). Links are provided above.

Step 3. Go to the Use page of this website.

Step 4. Click the dropdown box to select the version you're playing on then click the green Download button.

Step 5. Move the .jar files to the mods folder in the game directory (usually called .minecraft). If the folder doesn't exist, create one.
